Module: SignUp

Usage: http://52.11.98.82:3000/signup

Form will be POST whose type is multipart/form-data

feilds names {fb_id},{name},{dob},{user_gender},{req_gender}, {req_from_age}, {req_to_age}, {distance}, image array

Multipart/Multidata Form will be used to post fields and images

METHOD: POST

Parameters:
Name Type Description
fb_id Multipart/form-data facebook ID of the User
user_name Multipart/form-data Full Name of the User
dob Multipart/form-data Date of Birth of the User (Example: YYYYMMDD)
user_gender Multipart/form-data Gender of the User
req_gender Multipart/form-data required gender of the User
req_from_age Multipart/form-data Required Age Group Range
req_to_age Multipart/form-data Required Age Group Range
distance Multipart/form-data Required Area/range distance buffer area where anyone connect
image Multipart/form-data Multiple Picture format(jpg) more then l
Returns:
Results as RegisteredSuccessful or RegisteredUnSuccessful

Output Example: {"STATUS":"Registered Successful"}

Type
JSON
