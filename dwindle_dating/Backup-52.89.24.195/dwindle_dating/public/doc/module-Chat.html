Module: Chat

socket.io.connect to Usage: http://52.89.24.195:3000/Chat

socket.on('chat', function(fromUser, ToUser, status){} Client Emit chat event to Server and server will start to find the best matched Chat list user. A Socket will be created with the given fb_id

socket.on('getChatLog', function(json){} Client will use getChatLog event while server emit this event to client side and client will get JSON of all the chat history with to user pictures

socket.on('updatechat', function (user_name,data) {} client will receive chat with username

socket.on('sendchat', function (data) {} client will emit sendchat to server for chatting/sending messages to users

io.sockets.in(socket.room).emit('updatechat', socket.username, data ); updatechat with username, data as message which will update the chat to the clients

io.sockets.in(socket.room).emit('loggedout' ); logged out the users

io.sockets.in(socket.room).emit('disconnect' ); the event will occur when there is disconnection like no internet or other reason

socket.on('getChatLogForPage', function (from_user,to_user,current_page) client will emit this event with parameters: from_user,to_user,currentPageNo

socket.on('getChatLogForPageResult', function (json) {} client will receive next 20 chat log

User sendchat for messaging, data will be the meassage in text

updatechat to update messages on both side params:username, data in string

Results as JSON
